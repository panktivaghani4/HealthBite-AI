# HealthBite - AI Powered Smart Nutrition Recommendation System

HealthBite is an AI-powered smart nutrition recommendation system developed using Django. It helps users explore healthy food items, view nutritional information, and place food orders through an easy-to-use web interface.


## Features

- User Registration & Login
- Secure Authentication
- AI Powered Smart Nutrition Recommendation
- Healthy Food Recommendation System
- Product Management (Add, Update, Delete)
- Food Categories
- Product Search
- Product Details Page
- Shopping Cart
- Place Food Orders
- Order History
- Order Status Tracking
- Admin Dashboard
- User-Friendly Interface
- Responsive Design
- Image Upload for Products
- SQLite Database Integration

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Requirements

- Python 3.x
- Django
- pip

## Installation

```bash
git clone https://github.com/panktivaghani4/HealthBite-AI.git
cd HealthBite-AI
pip install -r requirements.txt
python manage.py runserver
```
